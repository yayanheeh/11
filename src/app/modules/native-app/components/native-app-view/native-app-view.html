2. Install

Open the downloaded Cloud-Player.dmg file

Drag and drop the Cloud-Player app into the Applications folder

3. Run Cloud-Player app

As this is a student project we don't have an Apple developer certificate. Therefore we could not sign the application. When you try to start it, macOS will tell you that the application is from an unidentified developer.

To open the app anyway locate Cloud-Player in the Applications folder. Control click the Cloud-Player app icon and then choose Open from the shortcut menu.

This has to be done only when you start Cloud-Player for the first time.

2. Install

Open the downloaded Cloud-Player.setup.exe file

As this is a student project we don't have an Windows developer certificate. Therefore we could not sign the application. The Windows Security Agent will prevent starting the installation process

To start the installation anyway click on the button "More Info" (1). A new button "Run anyway" will appear (2). Click on that button to start the installation process

The native application is open source as well
